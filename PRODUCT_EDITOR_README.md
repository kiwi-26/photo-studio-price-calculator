# 商品編集ツール (Product Editor Tool)

## 概要
このツールは、写真スタジオの価格計算アプリケーションに商品情報を追加・編集・削除する機能を提供します。

## 機能
- 商品一覧表示
- 商品の追加
- 商品の編集
- 商品の削除
- カテゴリ別フィルタリング
- 商品名での検索
- products.ts形式でのエクスポート
- 元データへのリセット

## 使用方法

### 1. 開発サーバーの起動
```bash
npm run dev
```

### 2. 編集ツールへのアクセス
- ブラウザで `http://localhost:5173` にアクセス
- 右上の紫色の鉛筆アイコンをクリックして編集モードに切り替え
- **注意**: 編集ツールはlocalhost環境でのみ利用可能です

### 3. 商品の管理

#### 新規商品の追加
1. 「新規追加」ボタンをクリック
2. 必要な情報を入力：
   - 商品名（必須）
   - カテゴリ（必須）
   - 説明（必須）
   - 価格（必須、0より大きい値）
   - 写真数（必須、0以上の値）
   - バリエーション（オプション）
3. 「追加」ボタンをクリック

#### 既存商品の編集
1. 商品一覧で編集したい商品の「編集」ボタンをクリック
2. 情報を修正
3. 「更新」ボタンをクリック

#### 商品の削除
1. 商品一覧で削除したい商品の「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

### 4. データのエクスポート
1. 「エクスポート」ボタンをクリック
2. 表示されたTypeScriptコードをコピー
3. 元の `src/assets/products.ts` ファイルに貼り付け

### 5. データのリセット
- 「リセット」ボタンをクリックすると、すべての変更が破棄され元の状態に戻ります

## 技術仕様

### 商品データ構造 (ProductType)
```typescript
interface ProductType {
  id: number | string;
  name: string;
  categoryId: string;
  description: string;
  photoCount: number;
  price: number;
  variation?: string;
}
```

### 利用可能なカテゴリ
- プリント (print)
- アルバム (album)
- アルバムプリント - 増えデジ (album-print-fuedegi)
- アルバムプリント - ベーシック (album-print-basic)
- 写真集 (photo-book)
- 写真台紙 - プレミア (photo-mount-premium)
- 写真台紙 - プレミア キャラクター (photo-mount-premium-character)
- 写真台紙 - ギフト (photo-mount-gift)
- 写真台紙 - デザイン ポエム付き (photo-mount-design-poem)

## セキュリティ
- 編集ツールはlocalhost環境でのみ動作します
- 本番環境では編集ボタンは表示されません
- ファイルの直接書き込みは行わず、画面上でのコピー&ペーストによる更新方式を採用

## 注意事項
- 編集内容はブラウザのセッション内でのみ保持されます
- ページをリロードすると変更内容は失われます
- 最終的な保存は手動でのファイル更新が必要です
- IDの重複を避けるため、新規商品には自動的に連番のIDが割り当てられます

## トラブルシューティング

### 編集ボタンが表示されない
- localhost環境で実行されているか確認してください
- ブラウザのアドレスバーが `localhost` または `127.0.0.1` で始まっているか確認してください

### フォームの入力エラー
- すべての必須項目（*マーク付き）が入力されているか確認してください
- 価格は0より大きい値を入力してください
- 写真数は0以上の値を入力してください

### エクスポートしたコードが正しく動作しない
- エクスポートされたコードが完全にコピーされているか確認してください
- TypeScriptの構文エラーがないか確認してください
