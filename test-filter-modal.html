🚀 テスト実行方法

1. ターミナルで以下のコマンドを実行:
npm run dev

2. ブラウザで http://localhost:5173 にアクセス

3. サイドバーの「フィルター」ボタンをクリック

4. モーダル内でフィルター条件を変更

5. 「反映」ボタンをクリックして動作確認
